NFL_Draftkings

Public Methods

  • get_scores(name=None, position=None, team=None, week=None, season=None)

    General method for getting score of any player, position or team in a specific season or week. If no arguments are specified, all players season stats from the latest season are returned.

    Arguments:
    • name: String
    • position: String
    • team: String
    • week: Integer
    • season: Integer

    Return: A list of player dictionaries.

  • get_weekly_scores(name=None, position=None, team=None, weeks=None, season=None)

    Method to get a player, position or team's scores week by week. If weeks is not specified it will be weeks 1-17.

    Arguments:
    • name: String
    • position: String
    • team: String
    • weeks: List or Tuple
    • season: Integer

    Returns: A list of player dictionaries.

  • get_top_performers(position=None, team=None, week=None, season=None, top=10):

    Method to get top performers for a given week or season, based on position, team or overall league.

    Arguments:
    • position: String
    • team: String
    • week: Integer
    • season: Integer
    • top: Integer

    Returns: A list of player dictionaries.
